A NUMBER GUESSING GAME

This is a simple python game where the computer generates a random number between 1 and 50, and the player tries to guess it. After each guess, the game tells the player whether the correct number is higher or lower of their guess. The game tracks how many guesses it takes to win and keeps a high score based on the lowest number of guesses or attempts it took. 

How to Run:

1. Python must be intalled on your computer.
2. Download the game file.
3. Open the terminal or command prompt. 
4. Navigate to the folder where the file is saved. 
5. Run the program using:
    python guessing_game.py

How to Play:

1. Enter each number between 1 and 50 when prompted.
2. The game will tell you if your guess is too high or too low.
3. Keep guessing until you get the correct number. 
4. After winning, you will see how many guesses it took. 
5. Try to beat your high score which is the lowest number of guesses. 
6. Choose whether to play again or quit.
